Everything you need to know about PDFForge.

Upload your file using the drag & drop area or file picker. Our server converts it to PDF using industry-standard tools and returns a download link. The process usually takes under 10 seconds.
Yes. All uploaded files are processed on our servers and automatically deleted after 30 minutes. We never read, share, or store your file contents beyond what's needed for conversion. Files are handled over HTTPS.
We currently support: JPG, JPEG, PNG (image to PDF), DOC, DOCX (Word to PDF), PPT, PPTX (PowerPoint to PDF), and PDF (merge and compress). We're adding more formats regularly.
The maximum file size is 50 MB per file. For merging multiple PDFs, the combined size of all files should not exceed 50 MB. If you need to convert larger files, consider compressing them first.
No account is required. PDFForge is completely free to use without registration. Just upload your file and convert.
Converting Word (.docx) and PowerPoint (.pptx) files requires LibreOffice to be installed on the server. LibreOffice is a free, open-source office suite that handles these formats faithfully. If you're self-hosting, install LibreOffice on your server with apt install libreoffice.
